# Nuclear Printing — Redesign Brief

_Source: Joe's two PDFs (Concept Overview + Interactive Overlay Concept), 2026-05-18._
_Decisions locked with CB this session._

## Vision
A living apparel-infrastructure ecosystem that feels like an extension of the
actual production floor — built to convert engagement into revenue. Not a
brochure, not a traditional apparel site.

## Tone locks
- **Must feel:** cinematic · immersive · operational · modern industrial · socially alive · constantly moving
- **Must NEVER feel:** loud · gimmicky · overwhelming · overanimated · startup-tech · cyberpunk
- **Governing principle:** restraint. Every glow, sound, transition earns its place.

## Conversion rule
Immersion strengthens conversion, never distracts. "Get a Quote" is the
persistent primary CTA, always one tap away at any depth.

## Brand locks (unchanged)
`#2db086` green · `#f2f2f2` · `#090909` · Antonio + Oswald + Archivo Narrow

## Page architecture (top → bottom)
1. **Nav** — clean cinematic bar: Apparel Production · Merchandising Systems · Resources · FAQ · Blog · Get a Quote. Logo = Home.
2. **Social Stories System** — under nav. IG-Stories-style row, production content only. Indicators softly pulse. "Production happening 24/7."
3. **Hero** — built for fast conversion. Button default = "Get a Quote."
4. **Apparel Production** — 4 categories: Design · Screen Printing · Embroidery · Direct to Film.
5. **Merchandising Systems** — operational infrastructure. Positions Nuclear as an apparel systems company.
6. **Blog** — AI-assisted intelligence layer, SEO-driven, visually fused into the world.

## Floating Button System (defining feature)
Persistent contextual-navigation object. NOT a chat widget. Soft inertia,
magnetic hover, controlled glow. Evolves by section:

| Section | Button text |
|---|---|
| Hero | Get a Quote |
| Apparel Production | Explore Apparel Production |
| Merchandising Systems | Explore Merchandising Systems |
| Production pages | See Today's Production |
| Embroidery | Inside Embroidery |
| Merch systems deep | Explore A Real System |
| About | Inside Nuclear |
| Interactive trigger | Try It Out |

## Interactive Layer (overlay)
On trigger, ~88% of screen smoothly expands into a deeper "operational room."
Page does NOT disappear, no redirect, environment faintly visible behind.
Houses: InkSoft store builders, design tools, apparel decks, live ordering,
production footage, case studies.

## Interaction filter
Every interaction must increase immersion / trust / retention / authority /
conversion / emotional connection — or it doesn't ship.

## Decisions locked (2026-05-18)
- **Button sound:** pneumatic "pshh" (industrial air-compressor release). Overrides the "raindrop pop" wording in the Overlay PDF.
- **Overlay size:** ~88%.
- **First build:** full homepage rebuild to the new architecture.
